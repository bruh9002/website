Config

setprefix - set the prefix of the bot.
setautorole - set an autorole (every member gets this role when he/she joins the server.)
setlogs - set the logging toa channel.
slowmode - sets the slowmode to x seconds.
slowmodeoff - turns slowmode off.
leveling (enable - disable) - turns the leveling on?off.
setwelcome - sets the welcome, leave channel to the mentioned channel.

Music
